About This Project

Overview


The Automated Prayer Times app was built to simplify the process of keeping track of daily prayer schedules. It automatically retrieves prayer times from your local mosque and delivers them to your email every morning.


The General Concept


  • Prayer times are scraped from verified mosque websites each day
  • A randomised hadith from Sahih Bukhari is generated daily
  • Emails are formatted with a table containing your city's prayer times and a hadith
  • Emails are sent out every morning to subscribed users at 6:30am GMT


Purpose Of This Project


This project was designed as a real-world automation tool in order to exercise my development skills, but more importantly, to benefit the ummah. I saw a real need for this application, and I believe (and hope) that one day it will aid Muslims across the UK in fulfilling their prayers in a country where services such as adhan (public prayer announcement) are not accessible.


How I Did It (for those curious)


This is a very simple, full-stack web application with a communicating front and backend.

  • The frontend is hosted on vercel
  • The backend is a Python script utilising the Flask framework, hosted on Fly.io
  • The script runs indefinitely, and is on a scheduler, constantly checking for time to send
  • Once assigned time (6:30am GMT) hits, the script executes and the emails are automatically formatted and sent
  • Frontend allows users to subscribe with their email, Backend receives subscriber's email and writes to a subscribers file

Future additions


  • Replacing subscribers file for subscribers database
  • Configuration for more UK cities, perhaps certain cities in different countries
  • Unsubscribe functionality — currently, users must contact me to be manually unsubscribed
  • Open to any other suggestions :)
